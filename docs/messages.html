param State aState

a state containing a substitution that makes the top goal true.

return

a stream of exactly one state, namely aState.

rtype

Srfi41Stream

The invariant that governs me can be stated as follows:

SBRAL new asState in: [:s | (true asGoal onState: s) car == s ] >>> true

or even better with the test case

GoalTest >> testSucceedInvariant 

	SBRAL new asState in: [ :s | 
		| ss |
		ss := true asGoal onState: s.
		self assert: ss car equals: s ].
	SBRAL new asState in: [ :s | 
		| ss |
		ss := s srfi41Singleton
			      zip: (true asGoal onState: s)
			      with: [ :a :b | a == b ].
		self assert: ss equals: { true } modulo: #asOrderedCollection ]
